## 1. Design intent (≤ 60 words)

- **Named direction:** **"Nebula Drift" — bioluminescent animated mesh.** Slowly-drifting neon magenta/cyan/violet mesh-gradient skies, soft glowing orbs, oversized gradient-filled kinetic headlines, particle starglow. Dreamy, vivid, club-poster-meets-deep-space — but premium and legible: neon is reserved for display + glow; body text sits on solid surfaces. Generous negative space keeps the glow luxurious, not loud.
- **Why this direction:** A premium spaceflight brand can sell *wonder* — a luminous, alive, almost-organic cosmos that feels like bioluminescent plankton in orbit.
- **Divergence from other variants this run:** v1 Event Horizon = cinematic dark *blue* + starfield realism; v2 Almanac = Swiss editorial monochrome; v3 Aurora = frosted *glass*; v4 Chromatica = *metallic chrome*. Nebula Drift is **soft neon light + color + motion** — no glass blur, no metal, no realism: pure luminous gradient atmosphere.

## 2. Inherited baselines (do not redescribe — link)

- **Theme contract** — light / dark / system, three-state, persisted, FOUC-safe, hydration-safe per **ADR-0004** + `custom_content-sdk-theme-hydration-pattern`. Tokens in Tailwind v4 `@theme` (`globals.css`), `@custom-variant dark`. Hex-literal tokens used via `var(--token)` directly — **never** `hsl(var(...))`-wrapped.
- **Accessibility floor** — WCAG 2.1 AA on **both** themes; semantic landmarks; visible focus; `prefers-reduced-motion` honored (FR-6 / NFR-3 / NFR-6).
- **Component prop boundary** — leaf = flat props, container = folder-of-children, nav excluded (**ADR-0005**). Invents no content shape a single field can't carry.
- **Forms** — Newsletter + Contact presentational-only with default/focus/error/success/empty states (**ADR-0006**).
- **NOT a Marketplace app — Blok is not used.** Nebula Drift defines its own bespoke neon token set; ignore all Blok enforcement.

## 3. Direction deltas (what Nebula Drift establishes — bespoke neon mesh system)

> POC is the visual source of truth. Below = tokens/devices the Lead Developer wires; composition lives in the POC.

### 3a. Type system
- **What:** Display = **Clash Display** (or **Fraunces** loaded weight-700 fallback) — a high-impact characterful grotesk, set OVERSIZED with a clipped gradient fill (`background-clip:text`). Body/UI = **Outfit** (geometric humanist sans, soft and modern); fallback `system-ui`. Numerals/stats = Outfit tabular.
- **Why:** Kinetic gradient-filled headlines are the signature; the body sans stays soft and round so UI feels friendly, not clinical.
- **Where in POC:** hero headline + all `PageHero` gradient-clip; `index.html § hero`.
- **Hook:** fluid `clamp()` `--step-*` scale; gradient headlines via `.kinetic` util (`background: var(--text-grad); -webkit-background-clip:text; color:transparent;`), animated `background-position` (reduced-motion → static).

- **Why two neon sets:** dark uses saturated vivid neon (glows on black); light uses **softened pastel** mesh blobs on tinted white so the mesh reads as luminous-dawn-aurora, not a recolored dark screen — and body text rides solid `--surface`/`--fg` for AA, never neon.
- **Hook:** mesh via layered CSS `radial-gradient`s on a `::before`; text-grad `--text-grad: linear-gradient(110deg, var(--neon-cyan), var(--neon-violet), var(--neon-magenta))`. Alpha glows via `color-mix(in oklch, var(--neon-x) N%, transparent)` — **never** `hsl(var(...))`.

### 3c. Spacing & layout rhythm
- **What:** Airy, poster-like — section rhythm `clamp(5.5rem, 13vh, 9rem)`; content max `~1180px` centered. Bands alternate: full-bleed mesh atmosphere vs contained solid-surface content. Big negative space around glowing focal elements so the neon never crowds.
- **Where in POC:** `index.html` band rhythm; gutters at 1440.

### 3d. Motion language ("drift")
- **What:** (1) **mesh-gradient drift** — 3-4 layered `radial-gradient` blobs slowly translate/scale on a 24-40s `@keyframes` loop (the hallmark); (2) **kinetic type** — headline gradient `background-position` animates left↔right ~8s; (3) hover: card lift + neon halo bloom (`box-shadow` color-mix); (4) scroll fade+rise reveal; (5) gentle orb-glow pulse on the hero focal orb.
- **Reduced-motion:** ALL collapse to static under `prefers-reduced-motion: reduce` — mesh frozen mid-frame, headline gradient static, pulse/reveal off. Single CSS guard at root + JS `matchMedia` (in `useEffect`, never render-body) gates any rAF.
- **Hook:** mesh + type animations are pure CSS `@keyframes` (no WebGL/images); `@media (prefers-reduced-motion: reduce){ *{animation:none!important} }`.

### 3e. Signature visual devices
Drifting neon mesh-gradient skies · oversized gradient-clipped kinetic headlines · a soft glowing focal "orb" in the hero (radial-gradient sphere with bloom) · particle/star dots (inline SVG, low-opacity, `aria-hidden`) · neon halo on hover cards · gradient hairline dividers · monochrome glyph icons (`✓`/`✕`, never emoji codepoints) for form state, tinted with `--accent-500`.

## 6. Accessibility notes

- **Contrast (verify on POC via `getComputedStyle`; AA ≥4.5:1 text / ≥3:1 large+UI):** Body text never sits on neon or mesh — it rides solid `--surface`/`--bg`. Dark `--fg #f4eeff` on `--surface #120c26` (~15:1, pass) and on `--bg #070512` (~17:1). Light `--fg #1c1233` on `--bg #f7f3ff` (~14:1) and `--surface #fff` (~16:1). Light `--accent-500 #8a2be0` on `--bg` for links/UI (verify ≥4.5:1; underline added so link affordance isn't color-only). Gradient-clipped neon headlines are LARGE display only (≥30px) — `--fg` solid fallback set before `background-clip` so a no-clip browser still reads. Final ratios captured on POC.
- **Keyboard flow:** logical tab order header → hero CTA → bands → footer; theme toggle + mobile drawer fully operable; drawer traps focus, closes on `Esc`, returns focus to trigger.
- **Reduced-motion:** `prefers-reduced-motion: reduce` → mesh frozen, kinetic type static, reveals/pulse off (POC honors; verify by toggling).
- **Forms:** errors via `aria-describedby` + `aria-invalid`; success `role="status"` live region. State icons monochrome glyphs (`✓`/`✕`) inheriting theme color — **no emoji codepoints**.
- **Imagery:** all mesh gradients, orb, particle SVG are `aria-hidden` decorative; content images carry alt text.
