## 1. Design intent (≤ 60 words)

- **Named direction:** **"Aurora" — iridescent glass over a luminous cosmos.** Frosted backdrop-blur panels, a holographic sheen that drifts across surfaces, glowing 1px gradient borders, and a bento-grid hero/feature mosaic. Pastel-neon accents (aurora cyan→violet→rose) over a deep luminous base. Apple-Vision-Pro elegance: shiny, premium, tactile — never gaudy.
- **Why this direction:** Premium spaceflight is *future-luxe*; glass + aurora light reads as next-gen craftsmanship the affluent traveller associates with the best hardware.
- **Divergence from other variants this run:** Opposite of v1 ("Event Horizon" — flat cinematic dark) and v2 ("Almanac" — hairline editorial Swiss). Aurora is the only **materials-led** direction: depth, translucency, and color-shift are the brand, expressed in a **bento mosaic** neither sibling uses.

## 2. Inherited baselines (do not redescribe — link)

- **Theme contract:** light / dark / system, mandatory, per ADR-0004 + `custom_content-sdk-theme-hydration-pattern`. FOUC preflight (`data-theme` set before paint) + `useSyncExternalStore` provider + mounted-flag inherited unchanged. Both themes **first-class** — §3b defines both glass + iridescence recipes.
- **Accessibility floor:** WCAG 2.1 AA contrast on both themes (PRD NFR-6). Glass legibility verified on POC in §6.
- **Component boundary:** flat-prop leaf / folder-of-children container per ADR-0005; presentational-only forms per ADR-0006. Bento tiles are a **layout** treatment of the existing components — they invent no new content shape a single Sitecore field can't carry.
- **NOT a Marketplace app → Blok is not used.** Bespoke Aurora token system below; ignore Blok enforcement. Hex-literal tokens consumed via `var(--token)` directly — never `hsl()`-wrapped (per `hsl_var_token_broken_with_hex_values`).

## 3. Direction deltas (what Aurora defines — bespoke iridescent-glass system)

> POC is the visual source of truth. Below = tokens/devices the Lead Developer wires; composition is shown in the POC, not re-prosed.

**3a — Type system.** Display = **Clash Display** (geometric, characterful, fashion-tech — distinctive at large sizes); fallback **"Space Grotesk"** then sans-serif. Body/UI = **Satoshi** (clean modern grotesk, premium feel); fallback **"Inter"**. Numerals/stats = Satoshi with `font-feature-settings:"tnum"`. Eyebrows uppercase, `letter-spacing:0.22em`, set in the accent gradient (`background-clip:text`). Fluid `clamp()` scale (ratio ≈1.28). *POC:* `index.html § hero`, `§ stats`. *Hook:* `--font-display`, `--font-sans`; load via `next/font` at port. (Clash/Satoshi self-host via Fontshare at port; POC uses Google `<link>` fallbacks Space Grotesk + Inter so it renders cleanly via `file://`.)

**3b — Color + material tokens (BOTH themes).** Luminous base + aurora accent triad + glass recipe. Accent named `--accent-500` per the `*-500` policy.

- **Glass recipe (the signature):** `background:var(--glass-fill)` + `backdrop-filter:blur(20px) saturate(150%)` + 1px gradient border (cyan→violet→rose, via `border-image` or a masked pseudo-element) + soft outer drop-shadow + a top inner-highlight (`--glass-stroke`). Dark glass = smoked translucency over the mesh; light glass = pearly milk-frost. *Hook:* `.glass` utility class; gradient border via `::before` masked ring (no `hsl()`).
- **Iridescent sheen:** a slow-drifting conic/linear holographic gradient overlay (`mix-blend-mode:overlay`, low opacity) on hero + featured bento tiles; shifts hue across the surface. Accent gradient = `linear-gradient(120deg, var(--accent-cyan), var(--accent-violet), var(--accent-rose))` reused for CTAs, eyebrows (text-clip), borders, focus glow.
- **Background:** layered radial **mesh gradient** (3 aurora blobs) behind everything + faint film grain (`aria-hidden` SVG noise) for tactility — generated only, no images.

**3c — Spacing + layout (BENTO).** 8px base scale. **Bento mosaic** is the defining layout: CSS Grid `grid-template-columns:repeat(12,1fr)` with tiles spanning varied col/row (e.g. hero tile 7-wide + tall, a value tile 5-wide, two 4-wide, a 6-wide stat tile). Consistent `gap:18px`, tile radius `28px`. Max content ≈1240px centered at 1440. Other bands (destinations, experiences, testimonials, newsletter) are also glass tiles in lighter bento arrangements. *POC:* `index.html § hero-bento`, `§ value-bento`.

**3d — Motion language ("shimmer").** (1) Page-load staggered rise+fade of bento tiles (`animation-delay` cascade); (2) iridescent sheen slow drift on hero/featured tiles (infinite, subtle); (3) hover = tile lifts (`translateY(-4px)`) + border-glow bloom + cursor-tracked specular highlight via a `--mx/--my` CSS var; (4) accent-gradient focus glow. **All** gated by `prefers-reduced-motion:reduce` → sheen/drift off, reveals instant, hover reduces to a static border brighten. *Hook:* single CSS reduced-motion guard + JS `matchMedia` (in `useEffect`/listener, never render-body) before starting any rAF/pointer loop.

**3e — Signature devices.** Frosted glass tiles · iridescent gradient 1px borders · holographic sheen drift · cursor-tracked specular highlight on tiles · aurora-gradient text eyebrows + CTAs · mesh-gradient + grain backdrop · bento mosaic. Form-state icons are monochrome glyphs (`✓` U+2713 / `✕` U+2715) that inherit theme color — **no emoji codepoints**.

## 6. Accessibility notes

- **Contrast (verify on POC via `getComputedStyle`, AA ≥4.5:1 text / ≥3:1 large+UI):** Text sits on the **opaque-equivalent** glass fill, not the raw mesh — dark `--fg #f3f5ff` over composited `--glass-fill` ≈ ≥13:1; light `--fg #0d1130` over light glass ≈ ≥14:1. Muted text kept ≥4.5:1. Aurora-gradient text used only for **large** eyebrows/headings (≥18.66px) — body never uses gradient text. CTA uses `--fg`-equivalent ink on the gradient fill, verified ≥4.5:1 at both ends of the gradient. *QA must re-capture actual composited ratios from the built POC — glass translucency is the risk; if any text pairing dips below AA, raise glass fill opacity for that tile.*
- **Keyboard flow:** logical tab order skip-link → header nav → theme toggle → hero CTA → bento tiles in reading order → footer; theme toggle + mobile drawer fully keyboard-operable; drawer traps focus while open, closes on `Esc`, restores focus to trigger. Focus = 2px aurora-gradient ring + 2px offset (never color-alone; visible over glass).
- **Reduced-motion:** `prefers-reduced-motion:reduce` → sheen drift off, tile-reveal cascade instant, cursor-specular off, hover = static border brighten only. POC ships this media query.
- **Screen-reader anatomy:** semantic landmarks (`header`/`nav`/`main`/`footer`); mesh-gradient, grain, sheen, specular, and decorative borders all `aria-hidden`; form errors `aria-describedby` + `aria-invalid`; success `role="status"` polite live region; state glyphs monochrome (`✓`/`✕`), never emoji codepoints.
