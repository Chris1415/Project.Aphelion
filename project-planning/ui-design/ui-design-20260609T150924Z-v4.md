## 1. Design intent (≤ 60 words)

- **Named direction:** **"Chromatica" — liquid-chrome aerospace instrument deck.** Brushed-aluminium and mirror-chrome surfaces with specular highlights, an anamorphic lens-flare hero, mission-control HUD overlays (reticles, monospaced telemetry ticks, corner readouts), and polished-steel CTAs with reflective gradients. Premium aerospace *hardware* — it gleams like machined metal.
- **Why this direction:** A spaceflight brand earns trust through engineered precision; chrome + telemetry says "real instrument," not "render."
- **Divergence from other variants:** v1 Event Horizon is atmospheric dark, v2 Almanac is flat editorial paper, v3 Aurora is translucent glass — **Chromatica is opaque metal**: specular, reflective, instrumented, hard-surfaced.

## 2. Inherited baselines (do not redescribe — link)

- **Theme contract** — light / dark / system, three-state, persisted, FOUC-safe, hydration-safe per **ADR-0004** + `custom_content-sdk-theme-hydration-pattern`. Tokens in Tailwind v4 `@theme` (`globals.css`), `@custom-variant dark`. Hex-literal tokens used as `var(--token)` directly — **never** `hsl(var(...))`-wrapped.
- **Accessibility floor** — WCAG 2.1 AA both themes; semantic landmarks; visible focus; `prefers-reduced-motion` honored (PRD FR-6 / NFR-3 / NFR-6).
- **Component boundary** — leaf = flat props, container = folder-of-children, nav excluded (**ADR-0005**). Invents no shape a single field can't carry.
- **Forms** — Newsletter + Contact presentational-only, full state set (**ADR-0006**).
- **NOT a Marketplace app → Blok is not used.** Bespoke chrome token system below; ignore Blok enforcement.

## 3. Direction deltas (what Chromatica establishes — bespoke metallic design system)

> POC is the visual source of truth; below = token/system hooks the Lead Developer wires.

**3a — Type system.** Display = **Orbitron** (geometric, machined, aerospace-instrument feel) at headlines/eyebrows. Body/UI = **Sora** (clean technical sans). **Telemetry/data = JetBrains Mono** for all HUD readouts, stat values, corner ticks, distances (`tnum`-aligned). Eyebrows uppercase, letter-spaced `0.34em`. Fluid `clamp()` scale. Google Fonts via `<link>` with system fallbacks. *POC:* `index.html § hero`, `§ stats`, all reticle readouts.
- *Hook:* `--font-display:"Orbitron"`, `--font-sans:"Sora"`, `--font-mono:"JetBrains Mono"`.

**3b — Color tokens (BOTH themes; consumed as `var(--token)` directly).** Metallic = layered gradients, not a flat hue. Each theme ships a *specular gradient recipe*, not just a colour.

| Token | Dark ("Gunmetal Deck") | Light ("Polished Titanium") | Role |
|-------|------------------------|------------------------------|------|
| `--bg` | `#070a0f` | `#e9edf2` | page base |
| `--surface` | `#11161e` | `#fbfcfe` | panel/card body |
| `--chrome-hi` | `#cfd8e3` | `#ffffff` | specular highlight stop |
| `--chrome-mid` | `#6b7889` | `#b9c4d2` | brushed-metal midtone |
| `--chrome-lo` | `#161c25` | `#9aa7b8` | metal shadow stop |
| `--fg` | `#eef3fa` | `#10151d` | primary text (on solid surface) |
| `--fg-muted` | `#94a1b4` | `#566276` | secondary text |
| `--accent-500` | `#34e0e8` | `#0a8f97` | HUD cyan — reticles, live data, CTA edge |
| `--accent-warm` | `#ff9a3c` | `#c25a07` | caution/solar accent (sparse) |
| `--line` | `#283342` | `#c2cdda` | hairline / reticle stroke |
| `--danger-500` | `#ff5c72` | `#c5283d` | form error |
| `--success-500` | `#39d39b` | `#0f8f63` | form success |

- **Metallic recipe (signature):** brushed surfaces = `linear-gradient` of `--chrome-hi → --chrome-mid → --chrome-lo` at a fixed angle + a faint repeating 2px gradient (brush grain) + SVG fractal-noise overlay at low opacity. Chrome CTAs add a moving specular band. **Both themes** carry the recipe so metal reads in light (silver-on-white) and dark (gunmetal-on-black). *POC:* `styles.css` `.chrome-*` utilities.
- *Hook:* text **always sits on `--surface`/`--fg`, never on raw mid-grey metal** (mid-greys fail AA). Accents via `*-500`; alpha via `color-mix(in oklch, …)` / SVG `stopOpacity` — never `hsl(var())`.

**3c — Spacing + layout rhythm.** 8px base (4/8/12/16/24/32/48/64/96/128). Content ≈1200px at 1440 with instrument-panel **margin rails** (thin vertical reticle lines + corner brackets framing the viewport). Sections are bolted "panels" separated by hairline rules + bracket ticks, not colour blocks. *POC:* `index.html § rails`, every section header.

**3d — Motion language ("specular sweep").** (1) Slow specular highlight sweep across chrome CTAs + hero wordmark; (2) anamorphic lens-flare streak drifts across hero on load; (3) HUD readouts type-in / tick like live telemetry; (4) hover = panel lifts + reticle brackets snap to corners + accent edge ignites. All **gated by `prefers-reduced-motion: reduce`** → sweeps/flare/ticks become static (flare = fixed streak, telemetry = final value, no panel transform). *POC:* honors reduced-motion (§6).
- *Hook:* single `@media (prefers-reduced-motion: reduce)` root guard zeroes all `--sweep`/`--flare` animations; rAF flare loop guarded by a `matchMedia` check in `useEffect` (never render body).

**3e — Signature devices.** Anamorphic lens-flare hero streak (CSS gradient + blur, no image). HUD corner brackets `⌐ ¬ L ⌙` (drawn as SVG/borders, not glyphs) on cards + viewport. Monospaced telemetry ticks (`ALT 412 km · Δv 11.2 km/s · LAT −24.6°`). Reticle crosshair behind stats. Brushed-metal CTA with running specular band + cyan ignition edge. Form state icons = monochrome glyphs (`✓` U+2713 / `✕` U+2715), **never emoji codepoints**.

## 6. Accessibility notes

- **Contrast (design targets; QA re-captures from built POC via `getComputedStyle`; AA 4.5:1 text / 3:1 large+UI):** text always on `--surface`/`--bg`, never on mid-grey metal. Dark `--fg #eef3fa` on `--bg #070a0f` ≈ 17:1; dark `--fg-muted #94a1b4` on `--bg` ≈ 7:1; dark `--accent-500 #34e0e8` on `--bg` ≈ 9:1 (text-safe). Light `--fg #10151d` on `--bg #e9edf2` ≈ 15:1; light `--fg-muted #566276` ≈ 5.5:1; light `--accent-500 #0a8f97` on `--surface #fbfcfe` ≈ 4.6:1 (AA text) — used for ≥18.66px/UI/edges; body-size cyan links carry an underline so affordance isn't colour-only. **Mid-grey metal tones are decorative only; no text rides them.**
- **Keyboard flow:** skip-link → header nav → theme toggle → main sections top-to-bottom → footer; 2px accent focus ring + 2px offset on every interactive element. MobileNav drawer traps focus, closes on `Esc`, returns focus to trigger.
- **Reduced-motion:** `prefers-reduced-motion: reduce` removes specular sweeps, lens-flare drift, telemetry type-in, panel lift → static final state. POC ships this media query.
- **Screen-reader anatomy:** decorative chrome/flare/reticles/brackets/telemetry ticks `aria-hidden`; stats use `<dl>` so value+label announce together; form errors `aria-describedby` + `aria-invalid`; success `role="status"` (polite). State icons monochrome glyphs that inherit theme colour — no emoji.
