## 1. Design intent (≤ 60 words)

- **Named direction:** **"Flux" — Nebula Drift, abstract-render edition.** Gooey morphing neon blobs drift behind the hero; abstract iridescent 3D-render / fluid imagery is the focal art (NOT literal space photos); kinetic gradient headlines run bolder; magnetic CTAs, scroll count-up stats, an infinite destination marquee, and flowing gradient borders add kinetic life. Premium, alive, legible.
- **Why this variation:** Same luminous Nebula-Drift cosmos, but "modern render + motion" — wonder you can feel move.

## 2. Inherited baselines (do not redescribe — link)

- **Base direction:** **v5 "Nebula Drift"** (`ui-design-20260609T150924Z-v5.md`). All of v5's intent holds unless overridden in §3. Token *names*, scale, band rhythm, theme contract, a11y floor are inherited.
- **Theme contract** — light / dark / system, three-state, persisted, FOUC-safe (inline preflight in `theme.js`), `localStorage` + `prefers-color-scheme`. Hex-literal tokens via `var(--token)` directly — **never** `hsl(var(...))`.
- **Accessibility floor** — WCAG 2.1 AA on **both** themes; landmarks; visible focus; `prefers-reduced-motion` honored.
- **Forms** — Newsletter presentational-only with default/focus/error/success/empty states.
- **NOT a Marketplace app — Blok is not used.** Bespoke neon token set; ignore Blok enforcement.

## 3. Direction deltas vs base v5 (what "Flux" changes)

> POC is the visual source of truth. Below = only what differs from Nebula Drift.

- **3a. Gooey morphing blobs (new hallmark).** A 4-blob field (cyan/magenta/violet + new `--neon-lime` accent) drifts behind the hero through an **inline SVG `#gooey` filter** (`feGaussianBlur` + `feColorMatrix` alpha-threshold) so blobs melt/merge organically. Replaces v5's static layered-`radial-gradient` mesh as the hero focal effect; the soft drifting mesh is retained underneath non-hero bands for atmosphere. *See `index.html § hero .flux-field`.*
- **3b. Abstract modern-render imagery (new).** Image-led media throughout uses **royalty-free abstract / iridescent / 3D-render / fluid photos** (Unsplash) as focal art — hero in a **morphing organic frame** (`border-radius` keyframe morph), destinations + experiences as `mix-blend:screen` render media. NOT literal space photos (that is sibling v5a). Token `--img-overlay` softens loud renders in light theme. *See `§ hero-art`, `§ destinations`, `§ experiences`.*
- **3c. Heavier motion (new).** **Magnetic hover** (`data-magnetic` → element follows cursor via `--mx/--my` translate) on CTAs + cards; **scroll count-up** on the stats band (`data-countup`, easeOutCubic); **infinite marquee** of 6 destination names (pause-on-hover); **flowing gradient border** on value cards (`.flowborder`); spinning brand mark. v5's reveal + kinetic-type + card-lift retained.
- **3d. Bolder kinetic headlines.** Gradient `background-size` 260% (v5: 220%) on a 7s cycle (v5: 9s) — more visible color travel.
- **3e. Token delta.** Adds `--neon-lime`, `--blob-opacity`, `--img-overlay`, `--media-blend`. All other tokens inherited from v5 (slightly deepened `--bg`/`--accent-500` for AA headroom — verified §6).

## 6. Accessibility notes (verified on POC via getComputedStyle)

- **Contrast — AA pass BOTH themes** (computed from tokens): `fg/bg` dark **17.82** / light **16.24**; `fg/surface` **16.74 / 17.74**; `fg-muted/bg` **9.03 / 6.92**; `fg-muted/surface` **8.48 / 7.56**; `accent-500/bg` **5.63 / 5.49**; `accent-500/surface` **5.29 / 6.00**. All ≥4.5:1. Body text rides solid `--surface`/`--bg`, never neon/mesh. Kinetic gradient headlines are LARGE display (≥30px) with solid `--fg` fallback when `background-clip:text` unsupported.
- **Reduced-motion (verified):** with `prefers-reduced-motion: reduce`, all 27 `[data-reveal]` are immediately shown, count-up jumps to final value, gooey blobs / marquee / magnetic / morph-frame / spin all freeze (`animation:none !important`, magnetic `transform:none`). No content trapped invisible.
- **Keyboard:** logical order; theme toggle + drawer operable; drawer focus-trap, `Esc`-close, focus-return-to-trigger (verified). Visible focus ring `--accent-500`.
- **Forms:** error via `aria-describedby` + `aria-invalid` + `role="alert"`; success `role="status"`. State icons monochrome glyphs `✓`/`✕` (theme-tinted) — **no emoji codepoints**.
- **Imagery:** blob field / mesh / orb / starfield / per-card mesh all `aria-hidden`; content render images carry descriptive `alt`. Marquee `aria-hidden` (duplicate decorative text).
